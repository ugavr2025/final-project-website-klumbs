Project Overview

Decider VR

Decider VR is a collaborative VR/MR application designed to help groups make shared decisions more easily. Users can add and react to options — like where to go or what to do — while an AI-powered NPC evaluates and ranks them based on criteria such as price, distance, and preferences.

The goal of Decision Maker is to create a more engaging and transparent group decision-making process, combining immersive XR environments, interactive UI panels, and intelligent AI analysis to help users reach fair and efficient outcomes.

Project Trailer

Project Details

AI-Driven NPC

Ranking & Recommendations

The AI NPC analyzes all submitted options along with user preferences and selected criteria. It ranks each option based on factors like distance, price, and personal preferences, then generates a clear recommendation with a short explanation.

Collaborative Experience

Multi-User Rooms & Avatars

Decision Maker supports private, multi-user rooms where friends join sessions to decide together. In VR, users are represented by avatars — either shared or randomly assigned.

UI System & Interaction

Menus, Criteria, and Reach Goals

The interface features a global start menu to begin sessions, restart, or close. Results are displayed visually with rankings, colors, or numerical scores. Each user can add or remove options and rank them.
